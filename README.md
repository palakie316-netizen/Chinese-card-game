# PaoHuZi Trainer (Starter Pack)

This is a **plain HTML/CSS/JS** starter pack for a phone-friendly **PWA** (installable web app) that teaches **跑胡子 (Pao Hu Zi)** with **voice guidance**.

## What’s included
- Single-page app navigation (Home / Lesson / Settings / Glossary)
- Voice guidance (Web Speech API)
  - Option B: English + 中文 + pinyin
  - Pronunciation: Slow vs Normal
- PWA setup (manifest + service worker + icons)
- GitHub Pages ready

## How to use (GitHub Pages)
1. Create a new GitHub repo (public is easiest for Pages).
2. Upload **all files and folders** from this starter pack (keep the folder structure).
3. Go to **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
4. Save. GitHub will show your site URL.

## Install on Android
1. Open your GitHub Pages URL in **Chrome**.
2. Tap the **⋮** menu.
3. Choose **Add to Home screen**.
4. Launch it like an app.

## Notes
- Some phones/browsers load voices slowly. If voice doesn’t speak at first, try the “Test Voice” button once.
- Chinese pronunciation quality depends on installed system voices.
